## What’s in This Repository

This repository contains **selected data analysis and data science projects** built with **Python**, **Jupyter notebooks**, **pandas**, **NumPy**, **Matplotlib**, and **Seaborn**. Each project includes datasets and notebooks for exploratory data analysis, visualization, and insights.

### Project Overview

| Project | Description |
|--------|-------------|
| **CO2 data** | CO₂ emissions analysis for Canada (CSV datasets and descriptions). |
| **E-commerce** | E-commerce and sales analysis: Amazon, international, and general sale reports with Python (pandas, Matplotlib, Seaborn). |
| **EDA_dbbank** | Exploratory data analysis of Deutsche Bank share prices: technical analysis (e.g. MACD), statistics, and visualizations. |
| **EDA_indian-election** | Indian election data EDA—national and state-level election datasets. |
| **EDA-Roller_coster_USA** | Roller coaster database and EDA for USA coasters. |
| **H2O2 and CH4 emission** | Background correlation analysis for H₂O₂ and CH₄ emissions (Excel data, Jupyter notebook). |
| **Online-sales** | Online sales data analysis (Jupyter notebook). |
| **xarray-fundamentals** | Introduction to **xarray** for multidimensional and netCDF-style scientific data (with example `.nc` file). |

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/KuchikiRenji/Data-analysis-projects.git
   cd Data-analysis-projects
   ```
2. **Set up a Python environment** (recommended: Python 3.8+ with `pip` or conda).
3. **Install dependencies**, for example:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter xarray
   ```
4. **Open any `.ipynb`** in Jupyter Lab/Notebook or VS Code and run the cells.
